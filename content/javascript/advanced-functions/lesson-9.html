Function Binding

When object methods are passed as callbacks (for example to setTimeout), the value of this can be lost.

JavaScript provides ways to preserve the correct this context using wrappers and the bind method.

Losing "this"

When a method is separated from its object, the reference to this is lost. 


let user = {
  firstName: "John",
  sayHi() {
    alert(\`Hello, \${this.firstName}!\`);
  }
};

setTimeout(user.sayHi, 1000);

Here setTimeout receives the function without its object, so this becomes undefined (or window in browsers).

Solution 1: Wrapper Function

We can wrap the method call inside another function. 


setTimeout(function() {
  user.sayHi();
}, 1000);


setTimeout(() => user.sayHi(), 1000);

However, if the variable user changes before the timeout executes, it may call the wrong object.

Solution 2: bind

The bind method creates a new function with a fixed this context. 


let boundFunc = func.bind(context);


let user = {
  firstName: "John"
};

function func() {
  alert(this.firstName);
}

let funcUser = func.bind(user);

funcUser();

Binding Object Methods


let user = {
  firstName: "John",
  sayHi() {
    alert(\`Hello, \${this.firstName}!\`);
  }
};

let sayHi = user.sayHi.bind(user);

sayHi();

setTimeout(sayHi, 1000);

Binding Arguments

The bind method can also fix arguments along with this. 


let bound = func.bind(context, arg1, arg2);

Partial Functions

When some arguments of a function are fixed, the new function is called a partially applied function. 


function mul(a, b) {
  return a * b;
}

let double = mul.bind(null, 2);

alert(double(3));
alert(double(4));

Another Example


let triple = mul.bind(null, 3);

alert(triple(3));
alert(triple(4));

Binding Multiple Methods

If an object has many methods that will be passed around, we can bind them all. 


for (let key in user) {

  if (typeof user[key] == "function") {

    user[key] = user[key].bind(user);

  }

}

Partial Without Changing Context

Sometimes we want to fix arguments but keep the original this context. 


function partial(func, ...argsBound) {

  return function(...args) {

    return func.call(this, ...argsBound, ...args);

  }

}

Example of Partial Function


let user = {

  firstName: "John",

  say(time, phrase) {

    alert(\`[\${time}] \${this.firstName}: \${phrase}\`);

  }

};

user.sayNow = partial(user.say, "10:00");

user.sayNow("Hello");

Summary

  • bind creates a new function with fixed this.
  • It allows passing object methods safely as callbacks.
  • It can also fix arguments for partial function application.
  • Partials are useful when the same argument is used repeatedly.
  • Custom partial helpers can preserve the original this context.
