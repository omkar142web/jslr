JSON Methods and toJSON

JSON (JavaScript Object Notation) is a standard format used to convert JavaScript objects into strings for storage or transmission and convert them back into objects.

Why JSON Exists

Object Serialization

Sometimes we need to convert a complex object into a string, for example to send it over a network, save it to a database, or print it in logs.

Manual Conversion Problem

We could manually implement a method like toString(), but that becomes difficult if objects become large or contain nested objects. JSON provides a built-in solution for this problem.

JSON.stringify()

JSON.stringify() converts a JavaScript object into a JSON string. 


let student = {
name: "John",
age: 30,
isAdmin: false,
courses: ["html","css","js"],
spouse: null
};

let json = JSON.stringify(student);

alert(typeof json);
alert(json);
The result is a string called a JSON-encoded or serialized object.

JSON Format Rules

Rule Description
Strings Must use double quotes
Keys Property names must also use double quotes
Allowed Types Objects, Arrays, Strings, Numbers, Booleans, null

JSON.stringify with Primitives


JSON.stringify(1);
JSON.stringify("test");
JSON.stringify(true);
JSON.stringify([1,2,3]);

Values Ignored by JSON

Some JavaScript values are skipped during JSON conversion.

Ignored Value Example
Functions object methods
Symbol keys Symbol("id")
undefined properties storing undefined 

let user = {
sayHi(){
alert("Hello");
},
[Symbol("id")]:123,
something: undefined
};

JSON.stringify(user);

Nested Objects


let meetup = {
title:"Conference",
room:{
number:23,
participants:["john","ann"]
}
};

JSON.stringify(meetup);

Circular References

JSON cannot convert objects that reference themselves. 


let room = {number:23};

let meetup = {
title:"Conference",
participants:["john","ann"]
};

meetup.place = room;
room.occupiedBy = meetup;

JSON.stringify(meetup);

JSON.stringify Options


JSON.stringify(value, replacer, space)
Parameter Description
value Value to encode
replacer Filter properties or transform values
space Pretty formatting indentation

Replacer Example


JSON.stringify(meetup,
["title","participants","place","name","number"]);

Replacer Function


JSON.stringify(meetup,function(key,value){

if(key == "occupiedBy") return undefined;

return value;

});

Formatting JSON


JSON.stringify(user,null,2);
The third parameter controls indentation for readable output.

Custom toJSON()

Objects may define a toJSON() method. JSON.stringify automatically calls it. 


let room = {
number:23,
toJSON(){
return this.number;
}
};

JSON.stringify(room);

JSON.parse()

JSON.parse() converts a JSON string back into a JavaScript object. 


let numbers = "[0,1,2,3]";

numbers = JSON.parse(numbers);

alert(numbers[1]);

Parsing Complex JSON


let userData = '{ "name":"John","age":35,"friends":[0,1,2] }';

let user = JSON.parse(userData);

alert(user.friends[1]);

Using reviver

The reviver function transforms values during parsing. 


let str = '{"title":"Conference",
"date":"2017-11-30T12:00:00.000Z"}';

let meetup = JSON.parse(str,function(key,value){

if(key=="date") return new Date(value);

return value;

});

alert(meetup.date.getDate());

Summary

  • JSON is a universal format for representing data.
  • JSON.stringify() converts objects to JSON strings.
  • JSON.parse() converts JSON strings back to objects.
  • JSON supports objects, arrays, strings, numbers, booleans and null.
  • Functions, symbols and undefined values are ignored.
  • The replacer argument customizes serialization.
  • The reviver argument customizes parsing.
  • If an object has toJSON(), it is used during serialization.
