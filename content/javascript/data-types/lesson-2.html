JavaScript Strings

Strings represent textual data in JavaScript. Internally they use UTF-16 encoding and can be written using single quotes, double quotes, or backticks.

Strings

What are Strings?

In JavaScript textual data is stored as strings. There is no separate type for a single character.

Internal Encoding

The internal format for strings is always UTF-16. It is not tied to the page encoding.

Quotes

Different String Quotes


let single = 'single-quoted';
let double = "double-quoted";
let backticks = \`backticks\`;

Single and double quotes behave the same. Backticks have additional capabilities.

Embedding Expressions


function sum(a, b) {
  return a + b;
}

alert(\`1 + 2 = \${sum(1, 2)}.\`);

Backticks allow expressions inside \${...}.

Multiline Strings


let guestList = \`Guests:
 * John
 * Pete
 * Mary
\`;

alert(guestList);

Single/Double Quotes Cannot Span Lines


let guestList = "Guests:
 * John";

This produces an error because traditional quotes do not support multiline strings.

Backticks appeared later in JavaScript and are more versatile. They also support tagged templates using the syntax: func\`string\`.

Special Characters


let guestList = "Guests:\\n * John\\n * Pete\\n * Mary";

The \\n character represents a newline. 


let str1 = "Hello\\nWorld";

let str2 = \`Hello
World\`;

alert(str1 == str2); // true
Character Description
\\nNew line
\\rCarriage return
\\'Single quote
\\"Double quote
\\\\Backslash
\\tTab
\\b \\f \\vLegacy control characters
All special characters begin with a backslash \\, known as an escape character.

Escaping Characters


alert(\`The backslash: \\\\\`);

Escaping Quotes


alert('I\\'m the Walrus!');
alert("I'm the Walrus!");

String Length


alert(\`My\\n\`.length); // 3

length is a property, not a function. Use str.length, not str.length().

Accessing Characters


let str = \`Hello\`;

str[0];
str.at(0);

str[str.length - 1];
str.at(-1);

Negative Index


let str = \`Hello\`;

alert(str[-2]); // undefined
alert(str.at(-2)); // l

Iterating Characters


for (let char of "Hello") {
  alert(char);
}

Strings are Immutable


let str = 'Hi';

str[0] = 'h';

Strings cannot be changed directly. 


let str = 'Hi';

str = 'h' + str[1];

alert(str); // hi

Changing Case


'Interface'.toUpperCase();
'Interface'.toLowerCase();

'Interface'[0].toLowerCase();

Searching for Substrings

indexOf()


let str = 'Widget with id';

str.indexOf('Widget');
str.indexOf('widget');
str.indexOf("id");

Searching Multiple Occurrences


let str = 'As sly as a fox, as strong as an ox';
let target = 'as';

let pos = 0;

while (true) {
  let foundPos = str.indexOf(target, pos);

  if (foundPos == -1) break;

  pos = foundPos + 1;
}

includes / startsWith / endsWith


"Widget with id".includes("Widget");

"Widget".startsWith("Wid");

"Widget".endsWith("get");

Getting a Substring

Method Description
slice(start,end) Selects from start to end
substring(start,end) Selects between start and end
substr(start,length) Selects length characters 

let str = "stringify";

str.slice(0,5);

str.substring(2,6);

str.substr(2,4);

Comparing Strings


'a' > 'Z';
'Österreich' > 'Zealand';
Strings are compared using their UTF-16 character codes.

localeCompare()


'Österreich'.localeCompare('Zealand');

Character Codes


"Z".codePointAt(0);
"z".codePointAt(0);

String.fromCodePoint(90);

Summary

  • Strings represent textual data using UTF-16 encoding.
  • Strings can use single quotes, double quotes, or backticks.
  • Backticks allow multiline strings and embedded expressions.
  • Strings are immutable.
  • Use slice or substring to extract substrings.
  • Use indexOf, includes, startsWith and endsWith to search text.
  • Use localeCompare for language-aware comparisons.
  • Useful methods include trim() and repeat().
