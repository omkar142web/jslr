JavaScript Array Methods

Arrays provide many built-in methods to manipulate, search, transform, and iterate over collections of data. In this chapter the methods are grouped by purpose to make them easier to understand.

Add / Remove Items

Basic Array Operations

We already know several methods that add or remove items from the beginning or the end of an array.

Method Description
arr.push(...items) Add items to the end
arr.pop() Remove item from the end
arr.shift() Remove item from the beginning
arr.unshift(...items) Add items to the beginning

splice()

Removing Elements

Arrays are objects, so technically we could remove items using delete. However this leaves empty holes in the array. 


let arr = ["I", "go", "home"];

delete arr[1];

alert(arr[1]); // undefined

alert(arr.length); // 3

The element was removed but the array length stayed the same.

The splice Method

The method arr.splice() is a powerful tool that can remove, insert, or replace elements. 


arr.splice(start[, deleteCount, elem1, ..., elemN])

It modifies the array starting at start, removes deleteCount elements, then inserts new elements.

Delete Element


let arr = ["I", "study", "JavaScript"];

arr.splice(1, 1);

alert(arr);

Replace Elements


let arr = ["I", "study", "JavaScript", "right", "now"];

arr.splice(0, 3, "Let's", "dance");

alert(arr);

Return Removed Items


let arr = ["I", "study", "JavaScript", "right", "now"];

let removed = arr.splice(0, 2);

alert(removed);

Insert Without Removing


let arr = ["I", "study", "JavaScript"];

arr.splice(2, 0, "complex", "language");

alert(arr);

Negative Index


let arr = [1,2,5];

arr.splice(-1,0,3,4);

alert(arr);

slice()

Create Subarrays

The method arr.slice() returns a new array containing a portion of the original array. 


arr.slice([start], [end])


let arr = ["t", "e", "s", "t"];

alert(arr.slice(1,3));
alert(arr.slice(-2));

Copy Array


let copy = arr.slice();

Calling slice with no arguments creates a full copy.

concat()

Combine Arrays

The method arr.concat() creates a new array by merging the current array with other arrays or values. 


arr.concat(arg1, arg2...)


let arr = [1,2];

alert(arr.concat([3,4]));
alert(arr.concat([3,4],[5,6]));
alert(arr.concat([3,4],5,6));

Array-like Objects


let arr = [1,2];

let arrayLike = {
  0:"something",
  length:1
};

alert(arr.concat(arrayLike));

Symbol.isConcatSpreadable


let arr = [1,2];

let arrayLike = {
  0:"something",
  1:"else",
  [Symbol.isConcatSpreadable]:true,
  length:2
};

alert(arr.concat(arrayLike));

forEach()

Iterate Elements

The method arr.forEach() runs a function for each element of the array. 


arr.forEach(function(item,index,array){
  // code
});


["Bilbo","Gandalf","Nazgul"].forEach(alert);


["Bilbo","Gandalf","Nazgul"].forEach(
(item,index,array)=>{
alert(\`\${item} is at index \${index}\`);
});

Searching in Arrays

indexOf, lastIndexOf and includes

These methods search for items in an array and behave similarly to their string counterparts.

Method Description
arr.indexOf(item, from) Returns index of item or -1 if not found
arr.lastIndexOf(item) Searches from right to left
arr.includes(item) Returns true if item exists 

let arr = [1, 0, false];

alert(arr.indexOf(0));
alert(arr.indexOf(false));
alert(arr.indexOf(null));

alert(arr.includes(1));

lastIndexOf


let fruits = ['Apple','Orange','Apple'];

alert(fruits.indexOf('Apple'));
alert(fruits.lastIndexOf('Apple'));

includes Handles NaN


const arr = [NaN];

alert(arr.indexOf(NaN));
alert(arr.includes(NaN));

find / findIndex

Search with Conditions

The method arr.find() searches an array using a function that returns a condition. 


let result = arr.find(function(item,index,array){
});


let users = [
{id:1,name:"John"},
{id:2,name:"Pete"},
{id:3,name:"Mary"}
];

let user = users.find(item => item.id == 1);

alert(user.name);

findIndex and findLastIndex

findIndex() returns the index instead of the element. findLastIndex() searches from right to left. 


let users = [
{id:1,name:"John"},
{id:2,name:"Pete"},
{id:3,name:"Mary"},
{id:4,name:"John"}
];

alert(users.findIndex(user => user.name == "John"));

alert(users.findLastIndex(user => user.name == "John"));

filter()

Multiple Matches

Unlike find(), the method filter() returns all matching elements as a new array. 


let results = arr.filter(function(item,index,array){
});


let users = [
{id:1,name:"John"},
{id:2,name:"Pete"},
{id:3,name:"Mary"}
];

let someUsers = users.filter(item => item.id < 3);

alert(someUsers.length);

map()

Transform Arrays

The method map() transforms each element and returns a new array with the results. 


let result = arr.map(function(item,index,array){
});


let lengths = ["Bilbo","Gandalf","Nazgul"]
.map(item => item.length);

alert(lengths);

The method arr.sort() sorts array elements in place. By default it sorts elements as strings. 


let arr = [1,2,15];

arr.sort();

alert(arr);

let countries=['Österreich','Andorra','Vietnam'];

countries.sort((a,b)=>a.localeCompare(b));

reverse()


let arr=[1,2,3,4,5];

arr.reverse();

alert(arr);

split() and join()

Convert String ↔ Array

The method split() converts a string into an array, while join() converts an array back into a string. 


let names='Bilbo, Gandalf, Nazgul';

let arr=names.split(', ');

for(let name of arr){
alert(\`A message to \${name}\`);
}

Limit Length


let arr='Bilbo, Gandalf, Nazgul, Saruman'
.split(', ',2);

alert(arr);

Split into Letters


let str="test";

alert(str.split(''));

join()


let arr=['Bilbo','Gandalf','Nazgul'];

let str=arr.join(';');

alert(str);

reduce() and reduceRight()

Reduce Array to a Single Value

The methods reduce() and reduceRight() calculate a single value based on the entire array. 


let value = arr.reduce(function(accumulator, item, index, array) {

}, initial);
Parameter Description
accumulator Result of previous function call
item Current array element
index Position in array
array The array itself

Sum Example


let arr=[1,2,3,4,5];

let result=arr.reduce(
(sum,current)=>sum+current,0
);

alert(result);

Without Initial Value


let arr=[1,2,3,4,5];

let result=arr.reduce(
(sum,current)=>sum+current
);

alert(result);
If the array is empty and no initial value is provided, reduce will throw an error.

Arrays are technically objects, so typeof cannot distinguish between objects and arrays.

Correct Method

Additional Useful Methods

Method Description
arr.some(fn) Returns true if any element matches condition
arr.every(fn) Returns true if all elements match condition

arr.copyWithin(target,start,end) Copies elements within array

Comparing Arrays

Avoid == Operator

Arrays should not be compared with == because they are objects. Two arrays are only equal if they reference the same object. 


alert([]==[]);
alert([0]==[0]);

Correct Comparison


function arraysEqual(arr1,arr2){

return arr1.length===arr2.length &&
arr1.every((value,index)=>
value===arr2[index]);

}

alert(arraysEqual([1,2],[1,2]));

Summary

  • Arrays provide many powerful built-in methods.
  • Use push, pop, shift, and unshift to add or remove items.
  • splice() can insert, remove, or replace elements.
  • slice() creates a copy or subarray.
  • concat() merges arrays.
  • indexOf, includes, find, and filter search arrays.
  • map() transforms arrays.
  • sort() and reverse() reorder arrays.
  • split() and join() convert between arrays and strings.
  • reduce() calculates a single value from the array.
  • Array.isArray() checks if a value is an array.
  • Never compare arrays using ==.
