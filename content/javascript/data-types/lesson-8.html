Object.keys, Object.values, Object.entries

JavaScript provides helper methods to iterate over the keys and values of plain objects. These methods make working with objects easier and allow transformations similar to array operations.

Iteration Across Data Structures

Common Iteration Methods

Many JavaScript data structures support iteration methods such as keys(), values(), and entries(). These methods are available in structures like Map, Set, and Array.

Objects Work Differently

Plain objects also support similar functionality, but the syntax is slightly different. Instead of calling methods on the object directly, we use static methods from the Object class.

Object.keys, Object.values, Object.entries

Method Description
Object.keys(obj) Returns an array of object keys
Object.values(obj) Returns an array of object values
Object.entries(obj) Returns an array of [key,value] pairs

Map vs Object Syntax

Feature Map Object
Call Syntax map.keys() Object.keys(obj)
Return Type Iterable Real Array
Object methods return real arrays for historical reasons, while Map methods return iterable objects.

Example


let user = {
  name: "John",
  age: 30
};

Object.keys(user);
// ["name","age"]

Object.values(user);
// ["John",30]

Object.entries(user);
// [["name","John"],["age",30]]

Looping Over Values


let user = {
  name: "John",
  age: 30
};

for (let value of Object.values(user)) {
  alert(value);
}

This loop prints the property values of the object.

Symbol Properties

Ignored by Default

Just like the for..in loop, the methods Object.keys(), Object.values(), and Object.entries() ignore properties whose keys are Symbols.

Accessing Symbol Keys

To get symbol properties, we can use Object.getOwnPropertySymbols(obj) or Reflect.ownKeys(obj).

Transforming Objects

Using Array Methods

Objects do not support array methods like map() or filter(). However, we can convert objects into arrays using Object.entries(), apply transformations, and convert them back with Object.fromEntries().

Example: Transform Object Values


let prices = {
  banana: 1,
  orange: 2,
  meat: 4
};

let doublePrices = Object.fromEntries(

  Object.entries(prices).map(
    entry => [entry[0], entry[1] * 2]
  )

);

alert(doublePrices.meat);

First we convert the object to an array, then transform it with map(), and finally convert it back into an object.

Summary

  • Object.keys(obj) returns an array of keys.
  • Object.values(obj) returns an array of values.
  • Object.entries(obj) returns an array of key/value pairs.
  • These methods return real arrays instead of iterables.
  • Symbol properties are ignored by default.
  • Object.entries() combined with Object.fromEntries() allows powerful object transformations.
