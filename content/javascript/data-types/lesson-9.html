Destructuring Assignment

Destructuring assignment is a special JavaScript syntax that allows us to unpack arrays or objects into multiple variables. It is extremely useful when working with objects, arrays, and function parameters.

Why Destructuring Exists

Two Core Data Structures

The two most used data structures in JavaScript are Object and Array. Objects store data by keys and arrays store ordered collections.

Selective Data Extraction

Often a function receives an object or array but only needs a few properties or elements. Destructuring allows us to extract only what we need directly into variables.

Array Destructuring


let arr = ["John","Smith"];

let [firstName,surname] = arr;

alert(firstName);
alert(surname);

The first element of the array goes into the first variable, the second element goes into the second variable.

Using with split()


let [firstName,surname] = "John Smith".split(" ");

alert(firstName);
alert(surname);

Destructuring Is Not Destructive

Destructuring does not modify the array. It simply copies the values into variables. 


let firstName = arr[0];
let surname = arr[1];

Ignoring Elements


let [firstName, , title] =
["Julius","Caesar","Consul","of the Roman Republic"];

alert(title);

Extra commas allow skipping unwanted elements.

Works with Any Iterable


let [a,b,c] = "abc";

let [one,two,three] = new Set([1,2,3]);

Destructuring internally uses iteration similar to for..of.

Assigning to Existing Objects


let user = {};

[user.name,user.surname] =
"John Smith".split(" ");

alert(user.name);
alert(user.surname);

Looping with Object.entries()


let user = {
name:"John",
age:30
};

for(let [key,value] of Object.entries(user)){
alert(\`\${key}:\${value}\`);
}

With Map


let user = new Map();

user.set("name","John");
user.set("age","30");

for(let [key,value] of user){
alert(\`\${key}:\${value}\`);
}

Swapping Variables


let guest="Jane";
let admin="Pete";

[guest,admin] = [admin,guest];

alert(\`\${guest} \${admin}\`);

Rest Operator


let [name1,name2,...rest] =
["Julius","Caesar","Consul","of the Roman Republic"];

alert(rest[0]);
alert(rest[1]);
alert(rest.length);

Default Values


let [name="Guest",surname="Anonymous"]=["Julius"];

alert(name);
alert(surname);

Using Functions as Defaults


let [name=prompt("name?"),
surname=prompt("surname?")]
=["Julius"];

Object Destructuring


let options={
title:"Menu",
width:100,
height:200
};

let {title,width,height}=options;

alert(title);
alert(width);
alert(height);

Renaming Variables


let options={
title:"Menu",
width:100,
height:200
};

let {width:w,height:h,title}=options;

alert(title);
alert(w);
alert(h);

Default Values for Objects


let options={title:"Menu"};

let {width=100,height=200,title}=options;

alert(title);
alert(width);
alert(height);

Rest Pattern with Objects


let options={
title:"Menu",
height:200,
width:100
};

let {title,...rest}=options;

alert(rest.height);
alert(rest.width);

Destructuring Without let


let title,width,height;

({title,width,height}=
{title:"Menu",width:200,height:100});
Parentheses are required because otherwise JavaScript interprets the braces as a code block.

Nested Destructuring


let options={
size:{width:100,height:200},
items:["Cake","Donut"],
extra:true
};

let{
size:{width,height},
items:[item1,item2],
title="Menu"
}=options;

alert(title);
alert(width);
alert(height);
alert(item1);
alert(item2);

Smart Function Parameters

Destructuring is extremely useful when functions have many optional parameters. Instead of remembering argument order, we can pass an object. 


function showMenu(
{title="Untitled",width=200,height=100,items=[]}
){
alert(\`\${title} \${width} \${height}\`);
alert(items);
}


function showMenu(
{title="Menu",width=100,height=200}={}
){
alert(\`\${title} \${width} \${height}\`);
}

Summary

  • Destructuring extracts values from arrays or objects into variables.
  • Array syntax: [item1,item2,...rest]
  • Object syntax: {prop:varName=defaultValue,...rest}
  • Works with nested structures.
  • Supports default values.
  • Useful for swapping variables.
  • Very helpful for function parameters.
