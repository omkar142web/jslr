JavaScript Numbers

In modern JavaScript there are two types of numbers: regular numbers stored in 64-bit IEEE-754 format, and BigInt for arbitrarily large integers.

Number Types

Regular Numbers

JavaScript numbers are stored in 64-bit IEEE-754 double precision format, also known as "double precision floating point". This handles both integers and decimals.

BigInt

BigInt represents integers of arbitrary length and is used when numbers exceed the safe integer range of ±(253 - 1).

Writing Large Numbers

Underscore Separator

let billion = 1_000_000_000;

Underscores improve readability and are ignored by JavaScript.

Scientific Notation

let billion = 1e9;
let value   = 7.3e9;

e multiplies the number by powers of 10.

Writing Small Numbers

let microSecond = 1e-6;  // 0.000001

Negative exponent divides the number by powers of 10.

Hex, Binary and Octal

let hex    = 0xff;        // 255
let binary = 0b11111111;  // 255
let octal  = 0o377;       // 255

Number Base Conversion

let num = 255;

num.toString(16);  // "ff"
num.toString(2);   // "11111111"
num.toString(8);   // "377"

The base can range from 2 to 36. Base 36 is useful for encoding long numeric identifiers as short strings.

toFixed()

let num = 12.34;

num.toFixed(1);  // "12.3"
num.toFixed(5);  // "12.34000"

The result of toFixed() is a string, not a number.

Floating Point Precision

0.1 + 0.2 === 0.3  // false
0.1 + 0.2          // 0.30000000000000004

Decimal fractions cannot always be represented exactly in binary. Use toFixed() or integer math to avoid precision issues.

NaN and Infinity

isNaN("text");         // true
isFinite(123);         // true
Number.isNaN(NaN);     // true
Number.isFinite(10);   // true

Infinity > 999999;     // true
-Infinity < -999999;   // true

parseInt and parseFloat

parseInt("100px");     // 100
parseFloat("12.5em");  // 12.5

parseInt with radix

parseInt("ff",   16);  // 255
parseInt("2n9c", 36);  // 123456

Math Object

Math.random();       // 0 to 1 (exclusive)
Math.max(3, 5, 10);  // 10
Math.min(1, 2);      // 1
Math.pow(2, 10);     // 1024
Math.sqrt(16);       // 4
Math.abs(-7);        // 7

Summary

  • Use e notation for very large or very small numbers.
  • JavaScript supports hex 0x, binary 0b, and octal 0o literals.
  • Use toString(base) to convert a number to a different base.
  • Use Math.floor, ceil, round, or trunc for rounding.
  • Floating point arithmetic can lose precision — use toFixed() when displaying.
  • Use parseInt and parseFloat to extract numbers from strings.
