JavaScript Date and Time

JavaScript provides the built-in Date object to store and manage dates and time. It allows us to measure time, work with timestamps, and retrieve or modify date components.

What is Date?

Date Object

The Date object stores both date and time together. It provides methods for retrieving and modifying components like year, month, day, hours, minutes, and seconds.

Common Uses

Dates are commonly used to store creation or modification times, measure performance, or display the current time.

Creating Dates

Constructor Description
new Date() Create a Date for the current date and time
new Date(milliseconds) Create a date from milliseconds since Jan 1, 1970
new Date(dateString) Create a date by parsing a string
new Date(year, month, ...) Create a date using components

Current Date


let now = new Date();

alert(now);

Timestamp Creation

A timestamp is the number of milliseconds that have passed since January 1st, 1970 (UTC+0). 


let Jan01_1970 = new Date(0);

alert(Jan01_1970);

let Jan02_1970 = new Date(24 * 3600 * 1000);

alert(Jan02_1970);

Negative Timestamp


let Dec31_1969 = new Date(-24 * 3600 * 1000);

alert(Dec31_1969);

Creating Date from String


let date = new Date("2017-01-26");

alert(date);
The time defaults to midnight GMT and is adjusted based on the local timezone where the code runs.

Creating Date from Components


new Date(2011,0,1,0,0,0,0);

new Date(2011,0,1);

Months are counted from 0 (January) to 11 (December). Missing parameters default to zero. 


let date = new Date(2011,0,1,2,3,4,567);

alert(date);

Accessing Date Components

Method Description
getFullYear() Get full year
getMonth() Get month (0-11)
getDate() Get day of month
getHours() Get hour
getMinutes() Get minutes
getSeconds() Get seconds
getMilliseconds() Get milliseconds
getDay() Get day of week (0=Sunday)
Avoid getYear(). It is deprecated. Use getFullYear() instead.

UTC Methods


let date = new Date();

alert(date.getHours());

alert(date.getUTCHours());

Special Methods

Method Description
getTime() Returns timestamp in milliseconds
getTimezoneOffset() Difference between UTC and local time zone

Setting Date Components

Method Description
setFullYear()Set year
setMonth()Set month
setDate()Set day of month
setHours()Set hours
setMinutes()Set minutes
setSeconds()Set seconds
setMilliseconds()Set milliseconds
setTime()Set timestamp

Autocorrection


let date = new Date(2013,0,32);

alert(date);


let date = new Date(2016,1,28);

date.setDate(date.getDate() + 2);

alert(date);

Date to Number


let date = new Date();

alert(+date);

Measuring Execution Time


let start = new Date();

for(let i=0;i<100000;i++){
let doSomething=i*i*i;
}

let end = new Date();

alert(end - start);

Date.now()

Date.now() returns the current timestamp without creating a Date object. 


let start = Date.now();

for(let i=0;i<100000;i++){
let doSomething=i*i*i;
}

let end = Date.now();

alert(end - start);

Date.parse()

Date.parse() reads a date string and returns the corresponding timestamp. 


let ms = Date.parse("2012-01-26T13:51:50.417-07:00");

alert(ms);


let date = new Date(Date.parse("2012-01-26T13:51:50.417-07:00"));

alert(date);

Summary

  • The Date object represents both date and time.
  • Months start from zero (January = 0).
  • Days of week start from zero (Sunday = 0).
  • Date objects auto-correct out-of-range values.
  • Subtracting two dates returns the difference in milliseconds.
  • Date.now() returns the current timestamp.
  • JavaScript timestamps use milliseconds.
  • For higher precision in browsers use performance.now().
