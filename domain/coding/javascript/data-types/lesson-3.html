JavaScript Arrays

Arrays are special objects designed to store ordered collections of data. They allow easy access, insertion, and removal of elements using numeric indexes.

Arrays

Ordered Collections

Objects store values using keys. But sometimes we need an ordered list where elements have positions: first, second, third, etc. Arrays are designed specifically for storing ordered collections.

Why Not Objects?

Objects do not provide convenient ways to manage order. We cannot insert properties between existing ones easily. Arrays solve this problem by using numeric indexes.

Declaration

Creating Arrays


let arr = new Array();
let arr = [];

Square brackets [] are the most commonly used syntax.

Array with Initial Values


let fruits = ["Apple", "Orange", "Plum"];

Accessing Elements


let fruits = ["Apple", "Orange", "Plum"];

alert(fruits[0]); // Apple
alert(fruits[1]); // Orange
alert(fruits[2]); // Plum

Replace Element


fruits[2] = "Pear";

Add Element


fruits[3] = "Lemon";

Array Length


let fruits = ["Apple", "Orange", "Plum"];

alert(fruits.length); // 3

Arrays Can Store Any Type


let arr = [
  "Apple",
  { name: "John" },
  true,
  function() { alert("hello"); }
];

alert(arr[1].name);
arr[3]();

Trailing Comma


let fruits = [
  "Apple",
  "Orange",
  "Plum",
];

Trailing commas make inserting or removing items easier.

Get Last Element with at()


let fruits = ["Apple", "Orange", "Plum"];

alert(fruits[fruits.length - 1]);

alert(fruits.at(-1));
arr.at(-1) returns the last element. Negative indexes count from the end of the array.

Stack and Queue

Queue (FIFO)

Queue follows the First-In-First-Out principle. Elements are added at the end and removed from the beginning.

Stack (LIFO)

Stack follows the Last-In-First-Out principle. Elements are added and removed from the end.

pop and push

pop()


let fruits = ["Apple", "Orange", "Pear"];

fruits.pop();

alert(fruits);

push()


let fruits = ["Apple", "Orange"];

fruits.push("Pear");

shift and unshift

shift()


let fruits = ["Apple", "Orange", "Pear"];

fruits.shift();

unshift()


let fruits = ["Orange", "Pear"];

fruits.unshift("Apple");

Multiple Elements


let fruits = ["Apple"];

fruits.push("Orange", "Peach");

fruits.unshift("Pineapple", "Lemon");

Internals

Arrays are actually special objects. Their indexes work like object keys, but JavaScript engines optimize them for fast ordered data storage.

Avoid treating arrays like objects by adding random properties or creating huge gaps between indexes.

Performance

Methods push() and pop() are fast because they work with the end of the array. Methods shift() and unshift() are slower because they must move all elements in memory.

Loops

for Loop


let arr = ["Apple", "Orange", "Pear"];

for (let i = 0; i < arr.length; i++) {
  alert(arr[i]);
}

for...of


let fruits = ["Apple", "Orange", "Plum"];

for (let fruit of fruits) {
  alert(fruit);
}

Avoid for...in


for (let key in arr) {
  alert(arr[key]);
}

This loop is designed for objects and may include non-numeric properties.

Length Property


let arr = [1,2,3,4,5];

arr.length = 2;

alert(arr);

Reducing length truncates the array.

Multidimensional Arrays


let matrix = [
  [1,2,3],
  [4,5,6],
  [7,8,9]
];

alert(matrix[0][1]);

Array toString


let arr = [1,2,3];

alert(arr);

alert(String(arr));

Comparing Arrays


alert([] == []);
alert([0] == [0]);
Arrays should not be compared using ==. They are objects and only equal if they reference the same object.

Summary

  • Arrays store ordered collections of data.
  • Create arrays using [] or new Array().
  • Elements are accessed using indexes starting from 0.
  • push() and pop() work with the end of the array.
  • shift() and unshift() work with the beginning.
  • for...of is the recommended way to iterate arrays.
  • Arrays are objects and are copied by reference.
  • Do not compare arrays using ==.
