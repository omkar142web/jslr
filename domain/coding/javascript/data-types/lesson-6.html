JavaScript Map and Set

Until now we used Objects for keyed collections and Arrays for ordered collections. However, JavaScript also provides two specialized structures: Map and Set.

Map

What is Map?

A Map is a collection of keyed data items similar to an object. The major difference is that Map allows keys of any type. Objects convert keys to strings, but Map preserves the key type.

Method Description
new Map() Create a new Map
map.set(key,value) Store value by key
map.get(key) Return value by key
map.has(key) Check if key exists
map.delete(key) Remove key/value pair
map.clear() Remove all elements
map.size Total number of elements

Map Example


let map = new Map();

map.set('1','str1');
map.set(1,'num1');
map.set(true,'bool1');

alert(map.get(1));
alert(map.get('1'));

alert(map.size);

Map keeps key types separate. Number 1 and string "1" are different keys.

Objects as Keys

Important Map Feature

Unlike objects, a Map can use other objects as keys. This makes it very useful when associating data with specific objects. 


let john = { name: "John" };

let visitsCountMap = new Map();

visitsCountMap.set(john,123);

alert(visitsCountMap.get(john));
Objects convert object keys to the same string "[object Object]", but Map keeps the object reference.

Map Key Comparison

SameValueZero Algorithm

Map compares keys using the SameValueZero algorithm. It behaves almost the same as strict equality ===, except that NaN is considered equal to NaN.

Method Chaining


map.set('1','str1')
   .set(1,'num1')
   .set(true,'bool1');

Each set() returns the map itself, allowing chained calls.

Iterating Over Map

Method Description
map.keys() Iterates keys
map.values() Iterates values
map.entries() Iterates key/value pairs 

let recipeMap = new Map([
 ['cucumber',500],
 ['tomatoes',350],
 ['onion',50]
]);

for(let vegetable of recipeMap.keys()){
 alert(vegetable);
}

for(let amount of recipeMap.values()){
 alert(amount);
}

for(let entry of recipeMap){
 alert(entry);
}

Map.forEach()


recipeMap.forEach((value,key,map)=>{
 alert(\`\${key}: \${value}\`);
});

Object.entries and Map

Create Map from Object

Object.entries(obj) returns an array of [key,value] pairs which can be used to create a Map. 


let obj={
 name:"John",
 age:30
};

let map=new Map(Object.entries(obj));

alert(map.get("name"));

Object.fromEntries

Convert Map to Object

The method Object.fromEntries() performs the reverse operation and converts key/value pairs into a plain object. 


let map=new Map();

map.set("banana",1);
map.set("orange",2);
map.set("meat",4);

let obj=Object.fromEntries(map);

alert(obj.orange);

Set

Unique Values Collection

A Set is a collection of values where each value may appear only once.

Method Description
new Set(iterable) Create a set
set.add(value) Add value
set.delete(value) Remove value
set.has(value) Check if value exists
set.clear() Remove all values
set.size Total values count

Set Example


let set = new Set();

let john = {name:"John"};
let pete = {name:"Pete"};
let mary = {name:"Mary"};

set.add(john);
set.add(pete);
set.add(mary);
set.add(john);
set.add(mary);

alert(set.size);

for(let user of set){
 alert(user.name);
}

Duplicate values are automatically ignored.

Iterating Over Set


let set=new Set(["oranges","apples","bananas"]);

for(let value of set){
 alert(value);
}

set.forEach((value,valueAgain,set)=>{
 alert(value);
});

Summary

  • Map is a collection of key-value pairs.
  • Map allows keys of any type, including objects.
  • Map preserves insertion order during iteration.
  • Object.entries() converts objects to Map format.
  • Object.fromEntries() converts Map to object.
  • Set is a collection of unique values.
  • Repeated values in a Set are automatically ignored.
  • Map and Set both support iteration using for..of.
