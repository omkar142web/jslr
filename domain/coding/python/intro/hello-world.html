Hello World

Welcome to Python. The first thing every programmer learns is how to print text to the screen.

Hello, World!

In Python, you do this using the print() function.

Your First Program

print("Hello, World!")

That's it! Unlike other languages, Python doesn't require semicolons at the end of statements, or complex boilerplate code just to get started.

Printing Multiple Items

You can print multiple items by separating them with a comma: 

print("Hello", "World", 2026)
