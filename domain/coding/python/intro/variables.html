Variables & Data Types

Variables are containers for storing data values. In Python, a variable is created the moment you first assign a value to it.

Creating Variables

x = 5
y = "John"
print(x)
print(y)

Python has no command for declaring a variable. You don't need to specify if `x` is an integer or a string. Python figures it out automatically based on the value assigned.

Built-in Data Types

Python has the following standard data types built-in by default:

  • Text Type: str
  • Numeric Types: int, float, complex
  • Sequence Types: list, tuple, range
  • Mapping Type: dict
  • Set Types: set, frozenset
  • Boolean Type: bool
  • None Type: NoneType

Getting the Data Type

You can get the data type of any object by using the type() function:

x = 5
print(type(x)) # Output: <class 'int'>

y = "Hello"
print(type(y)) # Output: <class 'str'>
